关于git的命令

git add . //添加所有修改的

git commit -m ‘hello’ // 备注

git push // 用于将本地分支的更新，推送到远程主机

git remote -v // 查询git下的项目名称

git log //查看提交记录

git clone + git地址 // 克隆

git remote add gitlab 链接地址 // 创建分支

5.配置环境变量

新增user的用户变量：

  • 变量：ANDROID_AVD_HOME
    值：C:\Users\user.android\avd
  • 变量：ANDROID_HOME
    值：C:\Users\user\AppData\Local\Android\Sdk
  • 变量：ANDROID_SDK_HOME
    值： C:\Users\user.android
  • 变量：ANDROID_SDK_ROOT
    值：C:\Users\user\AppData\Local\Android\Sdk
  • 变量：JAVA_HOME
    值： C:\Program Files\Java\jdk1.8.0_181\
  • 变量：JDK_HOME
    值：C:\Program Files\Java\jdk1.8.0_181\

在path变量中添加：

  • C:\Program Files\Java\jdk1.8.0_181\bin
